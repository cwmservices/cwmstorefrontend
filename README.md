CWMStore an ecommerce web app that promises to manage all your sales and here you can list almost every category of products and bunch of more features that includes:-

## Features

- Add To Cart
- Add To Whishlist
- Admin Can Add New Products
- Admin Can Add New Posts
- User Account And Authentication
- Order And Stripe Payment Gateway
- Authenticated User Can View Blogs And Order Products

## Pages

- Home
- Blog
- Sign Up
- Sign In
- About
- Contact
- Cart
- WhishList
- CheckOut
- User Profile
- Admin Profile
- Admin Account Login
- Admin Add Products
- Admin Add Posts
